# v0.1 Handoff Completion Evidence

Date: 2026-06-30 UTC

This artifact records the completed connector-factory milestone for
`rust-market-data`.

## Scope

The v0.1 handoff scope is public no-key market data only. It includes readiness
inventory, acceptance reports, mini drift audit, downstream consumer smoke, and
the first factory-built connector after the initial ready set.

Out of scope:

- private trading APIs
- account authentication
- signed requests
- balances
- orders
- execution
- a fully normalized cross-exchange event model

## Live Evidence

Run this when exchange-side evidence is needed:

```bash
cargo run -q -p acceptance -- drift-audit --json
```

The live release gate is `.github/workflows/live-acceptance.yml`. It is manual
and scheduled rather than a required pull-request gate because public exchange
APIs can drift, rate-limit, or fail independently of repository changes.

The downstream-style smoke command is:

```bash

```

It links against the handoff-ready connector crates through their public
interfaces and builds common market-data subscriptions without requiring private
exchange credentials.

## Next Connector Path

The Phase 2 next-venue PRD is GitHub issue `#16`. It selects OKX as the next
factory-built connector candidate and keeps `#1` open for the broader project
roadmap.

Future venues must follow the factory path documented in
`docs/connector-factory.md`:

1. status inventory
2. public REST proof
3. public WS proof
4. universe coverage or documented coverage scope
5. known quirks
6. README live proof
7. acceptance report
8. live drift-audit integration
9. downstream-style handoff proof

Do not mark a new crate `handoff-ready` because it has a WebSocket endpoint
string. It is ready only when the acceptance gates prove it.
